#### 💻 Technologies used 

<List items={['ReactJs', 'Electron', 'Prop-types', 'Redux', 'Immutable', 'Mocha/Enzyme']} />

In an Agile Scrum environment, we're delivering a digital transformation software platform for Ladbrokes / Coral staff to replace their Point of Sale system.

A large scale desktop software, front-end built on Electron using React. We're using the latest ES6 features with Redux & immutable, to name but a few. Thorough unit test coverage in place with over 10,000 unit tests.

Backend is a combination of Java & node js services hosted on AWS & local C# services on the local machine. Jenkins CI in place to run tests and package up Electron releases automatically from Git activity.
